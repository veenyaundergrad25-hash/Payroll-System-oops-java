# Payroll-System-oops-java
A Java-based Employee Payroll System developed using Object-Oriented Programming concepts.

## Features
- Add Full Time Employees
- Add Part Time Employees
- Salary Calculation
- Bonus Calculation
- Menu-Driven Program

## OOP Concepts Used
- Inheritance
- Method Overriding
- Encapsulation
- Constructors
- Classes and Objects

## Technologies Used
- Java
- VS Code

## How to Run

1. Compile the program:
   javac PayrollSystem.java

2. Run the program:
   java PayrollSystem
